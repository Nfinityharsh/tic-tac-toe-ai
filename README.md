# Tic Tac Toe AI

A simple Tic Tac Toe game where you play against an AI that uses the **Minimax algorithm** to make optimal moves.

## Features
- Play Tic Tac Toe against the computer
- AI uses Minimax algorithm
- Command-line interface

## Technologies
- Python
- Artificial Intelligence
- Algorithms

## How to Run

```bash
python tic_tac_toe.py
